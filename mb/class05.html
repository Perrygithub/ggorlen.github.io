Class 5: Introduction to JavaScript

Today, we'll wrap up HTML/CSS and ensure everyone is satisfied with how their page looks. Feel free to continue working on your website throughout the semester. We'll dive into JavaScript, which is the main focus of this class, and is the language we'll use to write our games.

The goal for this week is to work as a class to code a rock, paper, scissors game in JS and give it an HTML UI. During the course of the next few weeks, we'll learn more JS concepts, make a few simple games and solve various puzzles and problems in groups.


Today's topics:

  1. Recap HTML/CSS and make sure everyone is happy with their website. Discuss Bootstrap.
    • Check this out: an all-HTML Choose your own adventure game!
  2. Brief history of JavaScript, why it's worth learning, and what we'll be using it for.
  3. Introduction to JS basics
    • Adding JS scripts to HTML documents and using repl.it as an interpreter for running bits of code.
    • Literals, variables, strings, numbers, and the console.log() function.
      • 10 minutes: Break into pairs and solve variable puzzles.
    • Boolean operators and conditional statements.
      • 5 minutes: Solve conditional puzzles in pairs.
    • Functions and variable scope.
      • 15 minutes: Solve function puzzles in pairs.
  4. Before leaving, shut down your computer and plug it into a power source on the cart.
