  • Week 3: Hack week
    • Review, ask questions, and explore tangents.
    • Reduce smelly code; refactor and beautify your existing code.
    • Validate your HTML with the Nu HTML Checker.
    • Validate your CSS with the W3C CSS Validation Service.
    • Add more pages or directories to your website. For example, move "about.html" to "/about/index.html" and link it to main as follows:
      • Link inside "/about/index.html":
        <a href="../">Click to go back home</ a>
      • Link inside "index.html":
        <a href="./about/">Click for my about page</a>
      • In file structures, a period "." represents the current directory and two periods ".." represents the parent directory of the current directory.
    • Follow your classmates' Github accounts, look at their pages and read their code.
    • Collaborate on a webpage with a classmate.
    • Experiment with creating a bootstrapped website.
