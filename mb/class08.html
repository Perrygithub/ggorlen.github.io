  3. Implement a computer opponent with AI. We'll define a function called getComputerMove() which returns the computer's move as a string. There are many ways to decide how our AI behaves. Try a few on your own!
    • AI can choose randomly from the available options. Consider using a weighted random. Here's how to implement this:
      • We can get a random decimal between 0 and 1 using JS's Math.random() function. Try it out in the console! Almost all games use random numbers, so it's a very important function to remember. If we multiply the return value of Math.random() by the number of choices (in this case, 3) and cut off the decimal portion of the number, then we'll get either 0, 1, or 2 on each call. Try it:
      • var randomNum = Math.floor(Math.random() * 3);
      • Now, we can examine the random number we generated and say if (randomNum === 0) { computerCoice = "rock"; } and so forth.
      • However, the more elegant way to do this is to use an array, which is a variable which holds multiple values at once.
        • Declare an array like this: var choices = ["rock", "paper", "scissors"];
        • To access an element of this array, type the variable name followed by an index number in brackets: choices[2], for example, returns "scissors" (remember 0-indexing?).
        • Access a random element of the array using our random number routine to pick one of our three indexes: var computerCoice = choices[Math.floor(Math.random() * choices.length)];
          • choices.length is a property of our choices array which JS has provided. It returns the number of elements, or items, in the array, in this case, 3.
    • AI can look at the human player's choice and pick whichever beats (or loses against) that.
    • AI can cycle through a sequence of choices repeatedly using an array and a counter.
    • Use your imagination to come up with more ideas for AI behavior!
