Class 8: Project Planning

Let's do some preliminary project planning and keep working as a class on basic JS games!


Today's topics:

  1. Let's break into some groups and spend half an hour planning class projects. Here are some things to do and think about:
    • Brainstorm some possible project ideas (see the resources page for ideas).
    • What tools and skills will we need to accomplish these projects? Picking a specific project makes it easy to figure out which skills will be necessary to learn in our limited time.
    • What roles will each person play in the project? Programming is an obviously important part, but graphics, user interface, and planning/research may also be important tasks.
  2. Re-convene and do a little sample project with arrays and associative arrays.
  3. Try out some jQuery examples:
    • Draggable
    • Animate
  4. Next week, we'll learn about graphics in Javascript.
  5. Before leaving, please shut down your computer and plug it into a power source on the cart.
